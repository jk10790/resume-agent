# Resume Agent – Use Case

**Purpose:** Help candidates tailor resumes to job descriptions, evaluate fit, improve quality, and track applications. Single source of truth for the resume (e.g. Google Docs); tailored versions and evaluations on demand.

**User flows:** Extract JD from URL/paste → (optional) evaluate fit → tailor resume (light/medium/heavy) → (optional) improve quality → save/track.

**Constraints:** Do not fabricate facts (only use resume or user-confirmed list). Preserve candidate voice. ATS-aware formatting where relevant.
